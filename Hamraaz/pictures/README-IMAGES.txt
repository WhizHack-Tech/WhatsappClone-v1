HAMRAAZ IMAGES SETUP
====================

This folder should contain the following images for the website to display properly:

1. army-heroes.jpg
   - The main hero banner image (save the army heroes photo you shared here)
   - Recommended size: 800x400 pixels or similar aspect ratio
   - Format: JPG or JPEG

2. hamraaz-logo.png  
   - The Hamraaz portal logo (used in navigation AND login forms)
   - Recommended size: 120x120 pixels for login forms, will be scaled for navigation
   - Should contain the Hamraaz logo design (crossed swords and "HAMRAAZ" text)
   - Format: PNG (for transparency)
   - Note: Will be displayed at full size without any background circle

3. gov-india-logo.png
   - Government of India emblem/logo
   - Recommended size: 80x80 pixels
   - Format: PNG (for transparency)

4. azadi-ka-amrit-mahotsav.png
   - Azadi Ka Amrit Mahotsav celebration logo
   - Used in footer
   - Format: PNG (for transparency)

5. digital-india-logo.png
   - Digital India initiative logo
   - Used in footer
   - Format: PNG (for transparency)

6. new-icon-gif-2.jpg (Optional)
   - Small icon image used in scroll content cards
   - Size: 30x30 pixels
   - Currently referenced as "resou/new-icon-gif-2.jpg"
   - If not provided, will show as broken image but won't affect functionality

INSTRUCTIONS:
1. Save the army heroes image you uploaded as "army-heroes.jpg" in this folder
2. Find or create appropriate logos for hamraaz-logo.png and gov-india-logo.png
3. Refresh your browser to see the images appear on the website

The website will still work without these images, but you'll see broken image icons where the images should be displayed.
